Reveil Matin.






Fonctionalité :


- Affiche Heure et date du jour sur ecran lcd ( reglable ) (memoire permanante pile 2032) 
- Fonction alarme ( buzzer + animation ecran matrice )
- affichage date jour et mois sur matrice 


Design :

- Look tete de robot ( face avant ) 
- inscription similaire a cube mario avec police de carartere identique 


Materiel :

- arduino 
- ecran 16*2 lcd i2c
- 2 matrice 8*8
- rtc module ( permanance de l'heure ) 
- buzzer ( avec resistance ) 
- plaque circuit 
- bouton poussoir x4
- interrupteur on off 



Foctionalité possible avec le materiel ( non developpée ) :

- Ajout melodie avec buzzer 


Evolution possible :

- Module internet pour regalge automatique heure 
- possibiliter modification integration radio
